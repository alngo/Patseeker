# Patseeker

## Description

A library to seek candlestick pattern on key entry point.

The motive behind is to:
1. learn about domain modelling.
2. learn about Rust.
3. gain time on my daily technical analysis.
4. join my two passions, programming and trading.

## Installation

Compile to a .dll
``` bash
cargo build --target x86_64-pc-windows-gnu
```

## Usage

It is intended to be used along with Metatrader5
